# sewhail

ultra-minimal portfolio website with a terminal-inspired aesthetic.

## stack

- next.js 16
- typescript
- tailwind css 4
- shadcn/ui
- jetbrains mono

## features

- light/dark mode
- terminal-style design
- monospace typography
- ultra-minimal aesthetic
- responsive

## dev

```bash
npm install
npm run dev
```
